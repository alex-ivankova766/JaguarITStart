Здесь функция makeUser возвращает объект.

Каким будет результат при обращении к свойству объекта ref? Почему?

function makeUser() {
  return {
    name: "John",
    ref: this
  };
}

let user = makeUser();

alert( user.ref.name ); // Каким будет результат?

Предположение: здесь появляется условная рекурсия;
user.ref это и есть user, стало быть
user.ref.name соответствует user.name;
Результатом будет "John";

ЗАПОМНИТЬ: this принимает в качестве значения объект
тогда и только тогда, когда имеет место точечный вызов метода;
Отработало бы user.ref().name при условии, что ref()
является методом объекта, инициализированного функцией makeUser();
